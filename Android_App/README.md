# Mantak Android App

A fully native companion Android application for **Mantak (منطک)**, built using **Kotlin**, **Jetpack Compose**, and modern Android development practices.

## Features
- **Pure Kotlin & Jetpack Compose**: Beautifully styled using a high-contrast Brutalist theme that matches the React and iOS versions.
- **Bilingual Interface**: Seamlessly switch between English and Persian (RTL layout support).
- **Offline Logic Analytics & Server Connection**: Calls the same secure, production-ready logical analysis endpoint to identify cognitive and logical fallacies.
- **Durable Local Storage**: Saves your analysis history to SharedPreferences for permanent local persistence.
- **Programmatic Analytical Seal**: Renders a dynamic, administrative circular stamp showing the exact date and time of the analysis.
