# راهنمای اجرای منطک در لینوکس مینت ۱۹.۳ (Tricia) - نسخه ۳۲ بیت (x86/i386)

کاربر گرامی، نظر به اینکه سیستم‌عامل شما لینوکس مینت نسخه Tricia دسکتاپ XFCE معماری ۳۲ بیتی است، اجرای مستقیم برنامه‌های مدرن (به شکل فایل‌های باینری یکپارچه نظیر AppImage یا نسخه‌های جدید Electron) روی سیستم‌های ۳۲ بیت به دلیل پایان پشتیبانی متوقف شده است. با این حال، از آنجا که این یک نرم‌افزار تحت وب مبتنی بر Node.js و React است، شما به راحتی می‌توانید آن را از طریق سرویس‌دهنده محلی در مروگر خود (مثل فایرفاکس) اجرا کنید.

## پیش‌نیازها
شما برای اجرای بک‌اند برنامه (که به API متصل می‌شود) نیاز به `Node.js` دارید.
با باز کردن ترمینال (Ctrl+Alt+T) دستورات زیر را برای نصب نسخه متناسب ۳۲ بیتی وارد کنید:

```bash
sudo apt update
sudo apt install nodejs npm
```

*نکته: در مخازن Tricia (بنیان اوبونتو ۱۸.۰۴)، یک نسخه از Node.js سازگار با ۳۲ بیت وجود دارد که برای اجرای این نسخه باندل‌شده (server.cjs) کفایت می‌کند.*

## روش اجرا
۱. ابتدا در پوشه اصلی پروژه (یک سطح بالاتر از این پوشه)، دستور زیر را بزنید تا برنامه کامپایل و آماده شود (اینترنت وصل باشد):
```bash
npm install
npm run build
```

۲. یک فایل با نام `.env` در پوشه اصلی بسازید و کلید سرویس هوش مصنوعی خود را در آن قرار دهید:
```env
GEMINI_API_KEY="کلید_شما_در_اینجا"
```

۳. وارد همین پوشه (`Linux_Mint_32bit`) شوید و به فایل اجرای خودکار دسترسی بدهید:
```bash
chmod +x run.sh
chmod +x Mantak.desktop
```

۴. حالا با دو بار کلیک روی فایل `Mantak.desktop` یا اجرای `./run.sh` در ترمینال، بک‌اند برنامه در پس‌زمینه شروع به کار کرده و مروگر شما (احتمالاً Firefox موجود در مینت) تب منطک را باز خواهد کرد.

پایدار باشید!
